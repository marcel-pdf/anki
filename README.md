# LUXE - Luxury Furniture Studio

Минималистичный сайт премиальной мебельной студии, вдохновленный современным luxury дизайном.

## 🚀 Технологии

- **Next.js 14** - React фреймворк
- **TailwindCSS** - утилитарный CSS
- **Framer Motion** - анимации
- **GSAP** - продвинутые анимации при скролле
- **TypeScript** - типобезопасность

## 📦 Установка

```bash
npm install
```

## 🎯 Запуск

### Development
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

### Production Build
```bash
npm run build
npm run start
```
